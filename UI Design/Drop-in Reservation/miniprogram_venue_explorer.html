🚀 快速集成步骤
1. 创建页面文件

• 将上述四个文件（index.wxml, index.wxss, index.js, index.json）分别复制到对应位置

2. 添加图标资源
• 在项目根目录创建 images/icons/ 文件夹
• 将所有SVG图标文件放入该文件夹
• 图标文件大小建议控制在2KB以内

3. 配置页面路由
• 在 app.json 的 pages 数组中添加 "pages/venue-explorer/index"
• 如需作为tabBar页面，在 tabBar 配置中添加相应设置

4. API接口集成
• 修改 index.js 中的静态数据为实际API调用
• 建议使用 wx.request 或封装的网络请求库
• 注意处理网络异常和加载状态
⚡ 性能优化建议
图片优化
• 场馆图片使用 webp 格式，压缩率更高
• 设置 lazy-load 属性进行懒加载
• 使用小程序云存储CDN加速图片加载

数据优化
• 实现分页加载，每页建议10-20条数据
• 使用 setData 时只更新变化的数据
• 筛选和搜索结果进行防抖处理

渲染优化
• 使用 wx:key 为列表项设置唯一标识
• 长列表使用虚拟滚动或分页加载
• 避免在 wxml 中使用复杂的表达式计算
🔧 自定义配置
主题色彩
• 在 index.wxss 的 :root 中修改CSS变量
• 品牌色：--color-primary 和 --color-secondary
• 强调色：--color-accent 用于突出显示

筛选选项
• 在 index.js 的 data 中修改 gymTypeOptions 和 courseTypeOptions
• 可根据实际业务需求增加或删除筛选项
• 支持多级筛选和标签组合
